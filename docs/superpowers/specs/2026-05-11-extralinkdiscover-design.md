# ExtraLinkDiscover 设计文档

## 概述

Windows 桌面应用，用于爬取网站上的外部链接。用户输入目标网站和并发级别后，程序实时显示爬取进度和发现的外部链接。

## UI 布局

**单窗口设计（800x600）：**

```
┌─────────────────────────────────────────────────┐
│  目标网站: [________________________]            │
│  并发级别: [低▼]        [开始爬取] [停止]        │
│  过滤域名: [________________________] (可选)     │
├─────────────────────────────────────────────────┤
│  外部链接                    所在页面            │
│  ───────────────────────────────────────────    │
│  https://example.com       https://source.com   │
│  ...                                            │
│                                                 │
│                                                 │
├─────────────────────────────────────────────────┤
│  进度: ████████░░░░░░░░░░░ 45%                 │
│  已爬取: 45 页面    已发现: 123 链接            │
└─────────────────────────────────────────────────┘
```

## 功能需求

### 输入区
- URL 输入框：必填，验证格式
- 并发级别下拉框：低(5) / 中(10) / 高(20) / 极高(50)
- 过滤域名输入框：可选，支持泛域名(example.com)、指定域名、IP
- 开始按钮：启动爬取
- 停止按钮：终止爬取（进行中才可用）

### 结果区
- 表格两列：外部链接 URL、所在页面 URL
- 点击链接：在系统浏览器打开
- 导出按钮：保存结果到 CSV 文件

### 状态区
- 进度指示：显示"爬取中" / "已完成" / "已停止"
- 已爬取页面数：整数计数
- 已发现链接数：整数计数

## 技术方案

| 组件 | 技术选型 |
|------|---------|
| GUI框架 | eframe (egui的官方封装) |
| HTTP客户端 | reqwest |
| HTML解析 | scraper |
| 异步运行时 | tokio |
| 打开浏览器 | open crate |
| 序列化 | serde + csv |

## 数据流

1. 用户输入 URL → 点击开始
2. 异步爬取队列管理并发
3. 解析每个页面的 `<a href>` 外部链接
4. **过滤检查**：如果链接的域名/IP在过滤清单中，则跳过（不爬取、不统计）
5. 实时更新 UI（进度 + 链接列表）
6. 用户可导出 CSV 或点击链接打开浏览器

## 过滤规则

- 输入多个域名/IP时用换行分隔
- 支持泛域名匹配：输入 `example.com` 会匹配 `*.example.com`（包括 `example.com` 本身）
- 过滤发生在爬取前：过滤清单中的链接不会被爬取，也不会被统计为外部链接
- 默认不过滤（过滤框为空时）

## 错误处理

- 无效URL：提示"请输入有效的URL"
- 网络错误：记录错误，继续爬取其他页面
- 重复链接：去重后只显示一次
